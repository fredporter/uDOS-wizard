# v2.0.1 Orchestration Foundation

Wizard owns the `v2.0.1` starter orchestration surface for provider, publish,
and local-tool routing.

## Starter Surfaces

- assist routing through `provider-router`
- offline fallback routing through `local-shell`
- orchestration status via `/orchestration/status`
- MCP registry starter ownership remains in Wizard, but canonical abstraction
  still belongs to `uDOS-core`

## Scope

This is a small routing surface, not the full `v2.0.2` service layer.
