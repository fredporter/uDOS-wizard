# OK Diagrammer

## Purpose

Transform technical intent into structured visuals.

## Pipeline

`ascii -> md diagram -> teletext -> svg -> image prompt`

## Request classes

- `transformation`
- `multimodal`

## Budget intent

- preferred: `offline_only`
- escalation: `tier1_economy` / `tier2_premium` when justified

## Output classes

- ascii diagram
- markdown diagram block
- teletext block
- SVG
- image prompt pack

## Notes

Technical truth prefers SVG over image output.
